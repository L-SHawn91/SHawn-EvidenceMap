# Launch Plan

PUBLIC_STATUS: public-demo

## Goal

Publish a small, safe demo that proves the EvidenceMap concept before building accounts, billing, private uploads, or a full SaaS backend.

## Pre-launch

1. Confirm product name.
2. Use `dr.shawn91@gmail.com` as the commercial contact email and keep the GitHub pilot issue template as an alternate public intake path.
3. Run `bash scripts/public_safety_scan.sh`.
4. Run one live CLI query and save a polished sample output.
5. Review all public wording for manual-verification and non-clinical positioning.

## Launch Package

- `README.md`
- `web/index.html`
- `examples/outputs/endometrial-organoid-implantation.md`
- `docs/PUBLIC_RELEASE_CHECKLIST.md`
- Short screenshot or GIF

## Private First Share

Share privately with:
- 1 biomedical researcher
- 1 graduate student or postdoc
- 1 AI/product-minded reviewer

Ask:
- Do you understand what it does in 30 seconds?
- Would this save time before a review/grant/manuscript?
- Would you pay for a deeper evidence scan?
- What output format would you need?
