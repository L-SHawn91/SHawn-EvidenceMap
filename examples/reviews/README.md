# Review-file example

`example-review.json` documents the accepted schema. Replace every placeholder `paper_id` with an ID from the JSON output of the same EvidenceMap search before using it.

Review states are supplied by a person. EvidenceMap never auto-promotes a candidate row to `reviewed-support` or `reviewed-contradict`.

See [`../../docs/CLAIM_REVIEW_WORKFLOW.md`](../../docs/CLAIM_REVIEW_WORKFLOW.md).
